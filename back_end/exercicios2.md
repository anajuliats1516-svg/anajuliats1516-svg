1. O que é o Node.js ? Explique sua especificidade e por que ele é considerado um ambiente de execução e não uma linguagem de programação.
 Node.js é um ambiente de execução JavaScript assíncrono, de código aberto e multiplataforma, projetado para rodar no lado do servidor (back-end)

2. Qual a diferença entre Node.js e JavaScript executado no navegador ? Cite pelo menos duas diferenças .
A principal diferença entre Node.js e o JavaScript executado no navegador (browser) é o ambiente de execução e o propósito.
