**1.** O que é o Node.js?
E um ambiente de execução um código aberto e multiplataforma baseado no motor V8 do Chrome, que permite executar JavaScript no servidor backend.

**2.** O Node.js usa qual linguagem de programação?
Node.js utiliza principalmente JavaScript como linguagem de programação, permitindo o desenvolvimento de aplicações back-end

**3.** O Node.js roda no navegador ou fora dele?
O Node.js roda fora do navegador.

**4.** Para que serve o Node.js?
serve principalmente para executar código JavaScript.

**5.** Qual comando é usado para executar um arquivo Node.js no terminal?
arquivo Node.js no terminal é node <nome_do_arquivo.js>

**6.** O que é o **npm**?
Ele é utilizado para instalar, compartilhar e gerenciar dependências (bibliotecas, frameworks e ferramentas) em projetos de desenvolvimento JavaScript. 

**7.** O que significa a palavra **módulo** no Node.js?
 um módulo é uma unidade funcional de código, geralmente encapsulada em um arquivo JavaScript, que contém funcionalidades específicas, variáveis ou classes

**8.** O Node.js é mais usado no **front-end** ou no **back-end**?
JavaScript é considerado a linguagem de programação web mais popular do mundo, seja para back-end quanto para front-end. Como front-end, certamente a tecnologia mais utilizada é o Node.

**9.** O que é o arquivo **package.json**?
para armazenar e trocar dados de uma forma que seja legível por humanos e analisável por máquinas.

**10.** O Node.js é gratuito ou pago?
O Node.js é totalmente gratuito e de código aberto.
